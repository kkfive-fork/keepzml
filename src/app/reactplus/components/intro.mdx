React 知命境分为基础入门篇，与进阶尊享篇两部分。

进阶尊享篇主要围绕企业级项目中，如何更好的使用 React 进行开发。其中主要包括

+ 在基础版之上的一些高阶用法，通常这些高阶用法比较小众，只会在特定的高级场景中会使用到
+ 大型项目中的状态管理的思考
+ 大型项目中的代码架构设计思考
+ 大型项目中的开发质量思考
+ 大型项目中的开发效率思考
+ 大型项目中的更完整的性能优化思考

主要的叙述方式与 《React 19 尊享版》基本一致，会先从理论篇出发，然后在理论的基础之上，编写大量实践案例提供给大家学习。

与 《React 19 尊享版》不同的是，React 知命境更多的是从企业开发的角度出发，基于 React 传统的开发方式来学习如何更好的用好 React。而 React 19 则更加激进和前沿，与真实项目的开发方式可能存在脱轨的情况，所以 React 知命境不会过多的讲解 React 19 的新特性，而是从 React 传统的开发方式出发，来学习如何更好的使用 React。

注意：该专栏会按照基础版的推进进度，穿插更新。但与基础版不同的是，该专栏的内容，不会在公众号同步更新。
