import Cardruntime from 'components/Cardruntime'
import Pagination from 'components/Pagination'

### Pagination

默认用法

```js
<Pagination />
```
<Cardruntime>
  <Pagination />
</Cardruntime>


传入参数

```js
<Pagination total={22} current={18} />
```

<Cardruntime>
  <Pagination total={22} current={18} />
</Cardruntime>


onChange 事件获取 current

```js
<Pagination total={22} current={18} onChange={(i) => {console.log(i)}} />
```

<Cardruntime reload>
  <Pagination total={22} current={18} onChange={(i) => {console.log(i)}} />
</Cardruntime>
